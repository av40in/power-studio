Methodology & Formulas

Formula (unpooled / Welch):
n = (tα/2,df + tβ,df)² × (σ₁² + σ₂²) / Δ²
where Δ = |μ₁ − μ₂|, df = 2(n−1).
Formula (pooled):
n = (tα/2,df + tβ,df)² × 2σ² / Δ²
where σ² = (σ₁² + σ₂²) / 2 (average variance).
t-distribution correction:
Critical values use Student's t (not normal z), solved iteratively since df depends on n. This matches G*Power and prevents under-sizing at small n — important for typical animal group sizes.
Study designs:
Superiority — detect any difference (two- or one-sided).
Non-inferiority — show treatment is not worse by more than the margin (one-sided; effect size = assumed difference + margin).
Equivalence — show groups differ by less than ± margin (two one-sided tests / TOST; effect size = margin − |assumed difference|).
Bonferroni correction:
When comparing k groups, α is divided by k(k−1)/2 pairwise comparisons.
Dropout adjustment:
nadjusted = ceil(n / (1 − dropout%/100))
AEUC guidance (A/B/C interpretation):
A (minimum)Power = 50% — lowest justifiable n, high risk of missing effect B (optimum)Power = 80–90% — standard for most animal studies C (maximum)Power = 95% — maximum before study is over-powered
Based on AEUC/IACUC guidelines. Reference: Festing & Altman (2002). Guidelines for the Design and Statistical Analysis of Experiments Using Laboratory Animals.
